# Python-Flask_Notes
Flask practice notes <br/>
<br/>

## Important Packages to download:
* virtualenv-

                pip3 install virtualenv

* flask-

                pip3 install flask flask-sqlalchemy

<br/>

## Initial commands:
* create a virtual environment-

                virtualenv env

* activate the environment-

                source env/bin/activate
